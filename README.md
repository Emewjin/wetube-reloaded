# Youtube Clone - Wetube

## URL Plan
### Basic
- / => Home (Main Page)
- /join
- /login
- /search

### Users
- /users/:id => See user profile
- /users/logout
- /users/edit => Edit my profile
- /users/delete => Delete my profile

### Videos
- /videos/:id => see video
- /videos/:id/edit
- /videos/:id/delete 
- /videos/upload
